e18e MCP server

The official e18e MCP server, advising agents on modern and performant JavaScript practices. It flags inefficient or outdated npm packages and serves migration guides for moving off them.

The server is hosted over streamable HTTP at:

https://mcp.e18e.dev/mcp

Configure your agent

Claude Code

Run the following command, where [scope] is one of user, project or local: 

claude mcp add -t http -s [scope] e18e https://mcp.e18e.dev/mcp

Other clients

Refer to your client's documentation for remote (streamable HTTP) servers and use the URL above. Most clients accept a config of this shape: 

{
  "mcpServers": {
    "e18e": {
      "type": "http",
      "url": "https://mcp.e18e.dev/mcp"
    }
  }
}

A local stdio version is also available on npm as @e18e/mcp, with per-client setup instructions in the README.

What it exposes

  • npm-i-checker: checks an install command (npm i, pnpm add, yarn add, bun i) for packages with better alternatives.
  • code-checker: parses a source file and suggests replacements for the modules it imports.
  • lookup-replacement: looks up replacements by package name, replacement text or topic.
  • replacement-docs: a resource template (e18e://docs/{slug}) serving curated migration guides.
